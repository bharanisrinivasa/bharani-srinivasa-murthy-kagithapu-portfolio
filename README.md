## 💻 Running Locally

To run this project on your local machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/bharanisrinivasa/bharani-srinivasa-murthy-kagithapu-portfolio.git
   cd bharani-srinivasa-murthy-kagithapu-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open in browser:**
   Open `http://localhost:3000` (or the port specified by Vite) in your web browser.
